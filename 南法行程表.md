# 南法行程表

**檔案位置：** 南法旅行/南法行程表.html
**狀態：** 開發中
**類型：** 單一 HTML 互動式旅行規劃工具

## 概要
2026/06/29–07/15 南法蔚藍海岸 + 普羅旺斯 17 天 14 夜，兩位女生自由行。含行程、探索推薦、地圖、攻略、行李清單、筆記、同步功能。

## 技術
- 單檔 HTML（~700 行），無 build step
- Leaflet.js + OpenStreetMap 互動地圖
- localStorage 持久化（key: sf2026）
- Google Fonts: Inter + DM Serif Display
- 自訂 Nordic Vivid 主題：flat design、無漸層、無 box-shadow

## 色彩系統
- Nice 藍 `#2B8FBF`
- Arles 橘 `#E8734A`
- Aix 紫 `#8B5CF6`
- 綠色 OK `#10B981`
- 背景 `#FAFAFA`、文字 `#1E293B`

## 功能模組
1. 📅 行程 — 17 天每日時間軸，可從探索一鍵加入
2. 🗺️ 探索 — Nice / Arles / Aix 三區推薦卡片，可標記已去過＋收合
3. 📍 地圖 — 32 個標記點的互動地圖
4. 📋 攻略 — 部落客實戰筆記＋參考文章連結
5. 🧳 行李 — 7 大分類清單 + 自訂新增/刪除
6. 📝 筆記 — 自由記錄＋即時儲存
7. 🔄 同步 — 匯出/匯入/合併代碼，兩人免伺服器同步
